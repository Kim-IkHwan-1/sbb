# Chapter 2

### ORM과 JPM
- ORM : SQL을 사용하지 않고 데이터베이스를 관리할 수 있는 도구 <br>
- JPA : 스프링 부트는 JPA라는 인터페이스의 모음을 이용해 데이터베이스 관리

### 데이터베이스 구성 요소
데이터베이스는 기본적으로 2차원 표 형태로 저장하고 관리한다. <br>
기본키 : 테이블의 데이터가 중복되어 저장되지 않게 하는 것으로 어떤 열을 기본키로 설정하면 해당 열에는 동일한 값을 저장하지 못한다.
<br>
엔티티 : 데이터베이스 테이블과 매핑되는 자바 클래스


### 애너테이션
- @Id : id속성을 기본키로 지정한다
- @GeneratedValue : 데이터를 저장할 때 해당 속성에 값을 자동으로 1씩 증가시켜 저장
  - strategy = GenerationType.IDENTITY는 고유한 번호를 생설하는 방법을 지정하는 부분 
- @Column
  - length는 열의 길이를 설정할 때 사용
  - columnDefinition은 열 데이터의 유형이나 성격을 정의할 때 사용
